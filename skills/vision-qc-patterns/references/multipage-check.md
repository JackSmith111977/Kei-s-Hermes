# 多页抽样检查指南

## 问题

默认情况下，PDF 质检只检查第一页（首页）。但排版问题可能出现在：
- **中间页**：表格被分页切断
- **最后一页**：内容溢出边界
- **章节页**：页眉页脚状态变化

## 抽样策略

### 策略1：全量检查（文档 < 10 页）
```bash
pdftoppm -png -r 150 output.pdf /tmp/preview
# 检查每一张 preview-N.png
```

### 策略2：关键页抽样（文档 10-50 页）
- **第 1 页**：封面、标题
- **第 N/2 页**：中间内容
- **最后一页**：结尾、页码
- **每个章节的第一页**：检查页眉状态

### 策略3：随机抽样（文档 > 50 页）
- 每 10 页抽 1 页
- 必查：第 1 页、最后一页

## 自动化脚本

```bash
# 提取指定页为图片
pdftoppm -png -r 150 -f 1 -l 1 output.pdf /tmp/page_1
pdftoppm -png -r 150 -f 3 -l 3 output.pdf /tmp/page_3
pdftoppm -png -r 150 -f 5 -l 5 output.pdf /tmp/page_5
```

## 提取总页数
```bash
pdfinfo output.pdf | grep Pages
```

## Vision QC 多页检查 Prompt

```
请检查以下 N 张 PDF 页面预览图，排查以下问题：

1. 所有页面的页码是否连续正确？
2. 各章节标题的页眉是否正确更新？
3. 是否有任何页面的表格被分页切断？
4. 最后一页是否有内容溢出或空白过多？
5. 各页的行距、字号、间距是否一致？

逐页给出检查结果。
```
