## 错误处理最佳实践

### 1. 实现退避重试

```python
import time
import random

def retry_with_backoff(func, max_retries=3, base_delay=1):
    """带退避的重试"""
    for attempt in range(max_retries):
        try:
            result = func()
            if result.get("code") == 0:
                return result
            # 频率限制，等待后重试
            if result.get("code") == 99991400:
                delay = base_delay * (2 ** attempt) + random.uniform(0, 1)
                time.sleep(delay)
                continue
            return result
        except Exception as e:
            if attempt == max_retries - 1:
                raise
            time.sleep(base_delay * (2 ** attempt))
    return None
```

### 2. Token 自动刷新

```python
def call_with_token_refresh(api_func, auth: FeishuAuth):
    """调用 API，token 过期时自动刷新"""
    try:
        result = api_func()
        # token 过期
        if result.get("code") == 99991663:
            auth._token = None  # 清除缓存
            auth._expires_at = 0
            result = api_func()  # 重试
        return result
    except Exception:
        raise
```

### 3. 常见错误处理流程

```
API 调用
  ↓
code == 0? → 是 → 成功
  ↓ 否
code == 99991663? → 是 → 刷新 token → 重试
  ↓ 否
code == 99991400? → 是 → 退避重试
  ↓ 否
code == 99991671? → 是 → 提示申请权限
  ↓ 否
code == 99991672? → 是 → 检查资源 ID
  ↓ 否
其他错误 → 记录日志 → 联系飞书客服
```
