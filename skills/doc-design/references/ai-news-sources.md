## 搜索策略

### 按主题选择信源优先级:

1. **模型发布/产品** → 官方博客(1) + 顶级媒体(2)
2. **行业/商业** → 商业媒体(3) + 中文信源(6)
3. **学术/研究** → arXiv(4) + 实验室博客(1)
4. **政策/法规** → 政策信源(5) + 中文信源(6)
5. **技术/开发者** → 技术社区(8) + Newsletter(7)

### 时效性规则:
- 只收录 **3天内** 的新闻
- 每条新闻必须标注: 📰 报道时间 + 🔗 原文链接 + ⭐ 权威评分
- 日期格式: YYYY-MM-DD
- 超过3天的新闻标记为 `[过期]` 并排除

### 权威评分标准:
- ⭐⭐⭐⭐⭐: 官方来源、顶级同行评审、一手信息
- ⭐⭐⭐⭐: 专业科技媒体、知名研究机构
- ⭐⭐⭐: 行业媒体、知名博主
- ⭐⭐: 社区讨论、一般媒体
- ⭐: 自媒体、未经验证
