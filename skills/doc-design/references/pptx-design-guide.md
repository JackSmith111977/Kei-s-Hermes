## 一、2025-2026 PPT 设计趋势

### 1. 编辑风（Editorial Aesthetics）
- 受印刷杂志排版启发，大字体标题 + 高质量图片 + 精简文字
- 特点：大气、有冲击力、 timeless
- 应用：封面、章节页、重点强调页

### 2. 大胆排版（Bold Typography）
- 超大标题（48-72pt）占据页面 30-50%
- 字重对比：Ultra Light vs Heavy
- 字体作为设计元素，不只是信息载体

### 3. 深度与质感（Depth & Texture）
- 微妙阴影（soft shadows）创造层次感
- 触感纹理（tactile textures）增加视觉趣味
- 渐变色背景替代纯色

### 4. 克制配色（Restrained Color）
- 主色 1-2 种 + 中性色 + 1 个强调色
- 低饱和度色系（莫兰迪色系）显高级
- 蓝灰白最安全，少量橙/绿点缀

### 5. 极简留白（Generous Whitespace）
- 内容不超过页面 50%（比传统的 60% 更激进）
- 留白是设计元素，不是浪费空间
- 每个页面只传达一个核心信息

- **暗色主题**：亮色字 + 深色背景

## 三、字体系统（Typography）

### 中文字体搭配

#### 方案 A：经典搭配
- 标题：思源黑体 Bold / Medium
- 正文：思源黑体 Regular / Light
- 英文：Inter / Arial

#### 方案 B：现代搭配
- 标题：HarmonyOS Sans SC Bold
- 正文：HarmonyOS Sans SC Regular
- 英文：DM Sans / Poppins

#### 方案 C：优雅搭配
- 标题：Noto Serif SC Bold（衬线标题显优雅）
- 正文：Noto Sans SC Regular（无衬线正文易读）
- 英文：Georgia / Lora

### 字号层级（16:9 宽屏）
```
超大标题：   48-72pt（封面、章节页）
大标题：     36-44pt（页面标题）
中标题：     24-28pt（副标题）
正文：       16-20pt（正文内容）
辅助文字：   12-14pt（注释、说明）
最小文字：   10pt（脚注，尽量不用）
```

### 字体使用规则
- **不超过 2 种字体**（标题一种 + 正文一种）
- **字号差异 ≥ 4pt** 才能形成有效对比
- **行距 1.2-1.5 倍**
- **段落间距 ≥ 倍行距**
- **中英文混排时英文前后加半角空格**

## 四、布局系统（Layout System）

### 网格系统
- **基础单位**：0.5 inch 或 1cm
- **安全边距**：左右各 ≥ 1cm
- **内容区域**：不超过页面 50-60%

### 常见布局模式

#### 1. 居中对称式
```
┌─────────────────────────────┐
│                             │
│           [标题]            │
│                             │
│     [内容/图片/图表]        │
│                             │
│           [说明]            │
│                             │
└─────────────────────────────┘
```
适用：封面、重点强调、引用页

#### 2. 左对齐内容式
```
┌─────────────────────────────┐
│ [标题栏]                     │
│─────────────────────────────│
│ [内容]              [图片]  │
│                     [说明]  │
│                             │
│ [页码/页脚]                 │

```
适用：内容页、图文混排

#### 3. 卡片分栏式
```
┌─────────────────────────────┐
│ [标题栏]                     │
│─────────────────────────────│
│ [卡片1]  [卡片2]  [卡片3]   │
│                             │
│ [卡片4]  [卡片5]  [卡片6]   │
└─────────────────────────────┘
```
适用：功能介绍、特性列表

#### 4. 全图背景式
```
┌─────────────────────────────┐
│▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓│
│▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓│
│▓▓  [半透明文字框]         ▓▓│
│▓▓                          ▓▓│
│▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓│
└─────────────────────────────┘
```
适用：封面、章节过渡、情感页

### 对齐原则
- **每个元素必须与至少一个其他元素对齐**
- **使用参考线（Guides）辅助对齐**
- **避免居中对齐大段文字**（左对齐更易读）
- **相关元素靠近，不相关元素远离**

## 五、形状与装饰元素

### 常用形状
```python
# 圆角矩形（卡片背景）
MSO_SHAPE.ROUNDED_RECTANGLE

# 渐变色块（装饰背景）
MSO_SHAPE.RECTANGLE + gradient fill

# 圆形（图标容器、数字标注）
MSO_SHAPE.OVAL

# 箭头（流程指示）
MSO_SHAPE.RIGHT_ARROW

# 连接线
MSO_SHAPE.STRAIGHT_CONNECTOR
```

### 装饰技巧
- **半透明色块**：在图片上叠加半透明矩形，提升文字可读性
- **渐变色带**：页面顶部/底部的渐变色带，增加视觉层次
- **圆形/圆角容器**：包裹图标或数字，增加统一感
- **细分割线**：1-2pt 的浅色线，分隔不同区域
- **微阴影**：shape.shadow 增加深度感（但 python-pptx 支持有限，可用渐变模拟）

### 避免的装饰
- ❌ 花哨的边框
- ❌ 立体按钮效果
- ❌ 过多的图标装饰
- ❌ 彩虹渐变色
- ❌  Comic Sans 等不专业字体

## 六、动画与切换原则

### 推荐动画
- **Fade（淡入）**：最安全、最专业
- **Wipe（擦除）**：适合流程展示
- **Appear（出现）**：简洁直接

### 推荐切换
- **Fade（淡入淡出）**：最流畅
- **Push（推入）**：适合章节过渡
- **无切换**：最简洁

### 动画规则
- **整个 PPT 只用 1-2 种动画**
- **动画时长 0.3-0.5 秒**
- **避免弹跳、旋转、飞入等花哨效果**
- **列表逐项出现**（按段落，不是按行）

## 八、常见设计问题与解决方案

### 问题 1：页面太满
**原因**：想把所有信息都放上去
**解决**：
- 每页只传达一个核心信息
- 文字 ≤ 60 字/页
- 增加留白，内容不超过 50%

### 问题 2：颜色太杂
**原因**：每个元素用不同颜色
**解决**：
- 选定一套配色方案，全程使用
- 强调色只用于最重要的元素
- 灰色系（深灰、中灰、浅灰）是最安全的中性色

### 问题 3：字体太多
**原因**：想用不同字体创造变化
**解决**：
- 只用 1-2 种字体
- 通过字重（Bold/Light）和字号创造变化
- 不要在同一页面混用衬线和无衬线

### 问题 4：对齐混乱
**原因**：手动拖拽放置元素
**解决**：
- 使用参考线
- 左对齐是最安全的对齐方式
- 确保元素之间有明确的对齐关系

### 问题 5：图片质量差
**原因**：使用了低分辨率图片
**解决**：
- 图片分辨率 ≥ 1920x1080
- 不要拉伸小图片
- 使用高清图库（Unsplash、Pexels）

### 问题 6：缺乏层次
**原因**：所有文字大小一样
**解决**：
- 建立清晰的字号层级
- 标题比正文大 ≥ 8pt
- 使用颜色和字重区分层次

## 九、python-pptx 高级技巧

### 创建圆角卡片背景
```python
from pptx import Presentation
from pptx.util import Inches, Pt, Emu
from pptx.dml.color import RGBColor
from pptx.enum.shapes import MSO_SHAPE
from pptx.enum.text import PP_ALIGN, MSO_ANCHOR

def add_rounded_card(slide, left, top, width, height, 
                      fill_color, border_color=None, border_width=Pt(1)):
    """添加圆角矩形卡片"""
    shape = slide.shapes.add_shape(
        MSO_SHAPE.ROUNDED_RECTANGLE, left, top, width, height
    )
    shape.fill.solid()
    shape.fill.fore_color.rgb = fill_color
    if border_color:
        shape.line.color.rgb = border_color
        shape.line.width = border_width
    else:
        shape.line.fill.background()  # 无边框
    return shape

def add_gradient_background(slide, color1, color2):
    """添加渐变背景"""
    background = slide.background
    fill = background.fill
    fill.gradient()
    fill.gradient_stops[0].color.rgb = color1
    fill.gradient_stops[0].position = 0.0
    fill.gradient_stops[1].color.rgb = color2
    fill.gradient_stops[1].position = 1.0

def add_text_to_shape(shape, text, font_size=18, font_color=None, 
                      bold=False, alignment=PP_ALIGN.LEFT, font_name=None):
    """向形状添加格式化文字"""
    tf = shape.text_frame
    tf.word_wrap = True
    tf.auto_size = None  # 不自动调整大小
    p = tf.paragraphs[0]
    p.text = text
    p.font.size = Pt(font_size)
    p.font.bold = bold
    p.alignment = alignment
    if font_color:
        p.font.color.rgb = font_color
    if font_name:
        p.font.name = font_name
    return p

def add_styled_textbox(slide, left, top, width, height, 
                       title_text, body_text,
                       title_size=28, body_size=16,
                       title_color=None, body_color=None,
                       bg_color=None):
    """添加带标题和正文的文本框"""
    txBox = slide.shapes.add_textbox(left, top, width, height)
    tf = txBox.text_frame
    tf.word_wrap = True
    
    # 标题
    p = tf.paragraphs[0]
    p.text = title_text
    p.font.size = Pt(title_size)
    p.font.bold = True
    if title_color:
        p.font.color.rgb = title_color
    
    # 正文
    p2 = tf.add_paragraph()
    p2.text = body_text
    p2.font.size = Pt(body_size)
    p2.space_before = Pt(8)
    if body_color:
        p2.font.color.rgb = body_color
    
    if bg_color:
        fill = txBox.fill
        fill.solid()
        fill.fore_color.rgb = bg_color
    
    return txBox
```

### 创建三卡片布局
```python
def create_three_cards(slide, title, cards_data, 
                       card_width=Inches(3.5), card_height=Inches(4),
                       start_left=Inches(0.5), start_top=Inches(2),
                       gap=Inches(0.5),
                       bg_color=RGBColor(0xF8, 0xF9, 0xFA),
                       accent_color=RGBColor(0x44, 0x72, 0xC4)):
    """创建三卡片布局
    
    cards_data: list of dict with keys: 'title', 'content', 'icon' (optional)
    """
    # 页面标题
    title_box = slide.shapes.add_textbox(
        start_left, Inches(0.5), Inches(12), Inches(1)
    )
    tf = title_box.text_frame
    p = tf.paragraphs[0]
    p.text = title
    p.font.size = Pt(36)
    p.font.bold = True
    p.font.color.rgb = RGBColor(0x33, 0x33, 0x33)
    
    for i, card in enumerate(cards_data[:3]):
        left = start_left + i * (card_width + gap)
        
        # 卡片背景
        card_shape = add_rounded_card(
            slide, left, start_top, card_width, card_height,
            fill_color=bg_color
        )
        
        # 卡片顶部色条
        accent_bar = slide.shapes.add_shape(
            MSO_SHAPE.RECTANGLE, left, start_top, card_width, Inches(0.15)
        )
        accent_bar.fill.solid()
        accent_bar.fill.fore_color.rgb = accent_color
        accent_bar.line.fill.background()
        
        # 卡片标题
        title_box = slide.shapes.add_textbox(
            left + Inches(0.3), start_top + Inches(0.5),
            card_width - Inches(0.6), Inches(1)
        )
        tf = title_box.text_frame
        tf.word_wrap = True
        p = tf.paragraphs[0]
        p.text = card.get('title', '')
        p.font.size = Pt(22)
        p.font.bold = True
        p.font.color.rgb = RGBColor(0x33, 0x33, 0x33)
        
        # 卡片内容
        content_box = slide.shapes.add_textbox(
            left + Inches(0.3), start_top + Inches(1.8),
            card_width - Inches(0.6), card_height - Inches(2)
        )
        tf = content_box.text_frame
        tf.word_wrap = True
        p = tf.paragraphs[0]
        p.text = card.get('content', '')
        p.font.size = Pt(14)
        p.font.color.rgb = RGBColor(0x66, 0x66, 0x66)
```

### 创建章节过渡页
```python
def create_section_slide(prs, section_number, section_title, 
                          bg_color=RGBColor(0x1A, 0x20, 0x2C),
                          accent_color=RGBColor(0x4F, 0xD1, 0xC5)):
    """创建章节过渡页（深色背景 + 大标题）"""
    slide = prs.slides.add_slide(prs.slide_layouts[6])  # Blank
    
    # 深色背景
    background = slide.background
    fill = background.fill
    fill.solid()
    fill.fore_color.rgb = bg_color
    
    # 装饰线
    line = slide.shapes.add_shape(
        MSO_SHAPE.RECTANGLE, Inches(1), Inches(3.2), Inches(2), Pt(4)
    )
    line.fill.solid()
    line.fill.fore_color.rgb = accent_color
    line.line.fill.background()
    
    # 章节编号
    num_box = slide.shapes.add_textbox(
        Inches(1), Inches(2.5), Inches(3), Inches(1.5)
    )
    tf = num_box.text_frame
    p = tf.paragraphs[0]
    p.text = section_number
    p.font.size = Pt(72)
    p.font.bold = True
    p.font.color.rgb = RGBColor(0xFF, 0xFF, 0xFF)
    
    # 章节标题
    title_box = slide.shapes.add_textbox(
        Inches(1.5), Inches(4.2), Inches(10), Inches(2)
    )
    tf = title_box.text_frame
    tf.word_wrap = True
    p = tf.paragraphs[0]
    p.text = section_title
    p.font.size = Pt(36)
    p.font.color.rgb = RGBColor(0xE2, 0xE8, 0xF0)
    
    return slide
```

## 十、设计质量检查清单
