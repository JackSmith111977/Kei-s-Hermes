分析 [数据源] 数据，生成 [报告类型] 报告。

**触发词：** 分析 [数据源] 数据、生成 [报告类型]

---

## 分析流程

### Step 1：数据获取

1. 确认数据源位置：\`[数据源路径]\`
2. 验证数据完整性
3. 加载数据到分析环境

**数据验证：**
- 记录数是否合理
- 时间范围是否覆盖
- 字段是否完整

---

### Step 2：数据清洗

1. 处理缺失值
2. 去除重复记录
3. 标准化格式
4. 处理异常值

**清洗规则：**
| 问题 | 处理方式 |
|------|----------|
| 缺失值 | 填充/删除策略 |
| 重复 | 去重规则 |
| 异常 | 阈值过滤 |

---

### Step 3：分析执行

**分析维度：**
1. **描述性统计** — 均值、中位数、分布
2. **趋势分析** — 时间序列变化
3. **对比分析** — 分组对比
4. **相关性分析** — 变量关系

**工具选择：**
- Python (pandas + matplotlib)
- SQL 查询
- 统计检验
