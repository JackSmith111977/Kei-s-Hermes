## 检查清单

### 🔴 安全（高危）

- [ ] SQL 注入风险
- [ ] XSS 漏洞
- [ ] 敏感信息泄露（密钥、密码）
- [ ] 权限控制缺失
- [ ] 输入验证不足

### 🟡 质量（中危）

- [ ] 函数过长（> 50 行）
- [ ] 嵌套过深（> 4 层）
- [ ] 命名不清晰
- [ ] 缺少错误处理
- [ ] 代码重复
- [ ] 缺少注释/文档

### 🟢 建议（低危）

- [ ] 性能优化机会
- [ ] 可读性改进
- [ ] 类型注解缺失
- [ ] 测试覆盖不足
