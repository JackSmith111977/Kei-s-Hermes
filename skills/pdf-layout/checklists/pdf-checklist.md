# PDF 排版设计检查清单

生成 PDF 后，按以下清单逐项检查：

## 元数据
- [ ] 版本号已定义（`__version__` 变量）
- [ ] 作者信息正确
- [ ] 标题/主题/关键词已设置
- [ ] 创建日期正确
- [ ] 文件名包含版本号

## 中文渲染
- [ ] 所有中文正常显示（无黑色方块）
- [ ] 使用 `UnicodeCIDFont('STSong-Light')`
- [ ] Table 单元格已用 Paragraph 包装
- [ ] 特殊字符已转义（`&` `<` `>`）

## 排版布局
- [ ] 字体种类 ≤ 2 种
- [ ] 字号层级清晰（H1 > H2 > H3 > 正文，差异 ≥ 4pt）
- [ ] 段落间距 > 行间距
- [ ] 颜色对比度 ≥ 4.5:1
- [ ] 留白充足（内容不超过页面 50%）

## 表格
- [ ] 表头有底色
- [ ] 斑马纹交替
- [ ] 边框统一
- [ ] 单元格内边距 3-4mm
- [ ] 垂直居中对齐

## 页眉页脚
- [ ] 页眉显示书名/章节名
- [ ] 页脚显示页码
- [ ] 首页可不同（封面无页眉）
- [ ] 分隔线可见

## 目录
- [ ] 目录条目完整
- [ ] 页码准确
- [ ] 层级缩进正确

## 图片
- [ ] 图片清晰（分辨率 ≥ 150dpi）
- [ ] 自适应缩放（不超出页面）
- [ ] 有标题/说明
- [ ] 居中对齐

## 代码块
- [ ] 等宽字体
- [ ] 背景色区分
- [ ] 行高适当

## 分页
- [ ] 标题不被截断（KeepTogether）
- [ ] 表格跨页时表头重复
- [ ] 章节从新页开始（PageBreak）

## 文件
- [ ] 文件大小合理（< 50MB）
- [ ] 可在标准 PDF 阅读器打开
- [ ] 书签/大纲可导航
