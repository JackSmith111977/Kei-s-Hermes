# PPT 设计完整指南

> 基于 2025-2026 年行业最佳实践整理
> 来源：Microsoft, PitchWorx, VerdanaBold, PPTWiki 等

## 一、CRAP 四大设计原理

由 Robin Williams《The Non-Designer's Design Book》(1994) 提出，至今仍是 PPT 设计的基石。

### 1. Contrast（对比）
- 不同元素必须**显著不同**，而非"稍微不同"
- 通过字号、颜色、字体、粗细制造对比
- 最重要的信息用最大的字号+最粗的字重+最突出的颜色

### 2. Repetition（重复）
- 跨页重复视觉元素创造统一感
- 重复的内容：配色方案、字体、图标风格、卡片圆角、间距
- 一致性是专业度的信任信号

### 3. Alignment（对齐）
- 没有元素是随意放置的
- 使用8pt网格系统，所有间距为8的倍数
- 推荐使用 `View > Grid and Guides > Smart Guides`

### 4. Proximity（亲密性）
- 相关的元素放一起，不相关的分开
- 一组内的间距小于组间的间距
- 视觉关系反映逻辑关系

## 二、配色体系

### 色轮基础
- **互补色**：色轮上相距180°（如蓝+橙），强烈的视觉冲击
- **类似色**：色轮上相邻的颜色（如蓝+蓝绿），和谐统一
- **三角配色**：色轮上等距120°，丰富而平衡
- **分裂互补**：一个颜色 + 互补色的两侧颜色

### PPT 常用配色方案

| 方案 | 主色 | 辅助色 | 强调色 | 背景 | 场景 |
|------|------|--------|--------|------|------|
| 经典商务蓝 | #2B579A | #4472C4 | #ED7D31 | #FFFFFF | 工作汇报 |
| 现代渐变紫 | #667eea | #764ba2 | #f093fb | #FFFFFF | 产品发布 |
| 深色科技风 | #4FD1C5 | #81E6D9 | #F6AD55 | #1A202C | 技术分享 |
| 莫兰迪高级灰 | #8B9DC3 | #D4A5A5 | #C9B99A | #FAFAFA | 学术汇报 |
| 午夜行政 | #1E2761 | #CADCFC | #FFFFFF | #FFFFFF | 董事会 |
| 珊瑚活力 | #F96167 | #F9E795 | #2F3C7E | #FFFFFF | 创意提案 |
| 森林苔藓 | #2C5F2D | #97BC62 | #F5F5F5 | #F5F5F5 | 环保主题 |
| 深色木炭 | #36454F | #F2F2F2 | #212121 | #F2F2F2 | 高端品牌 |

### 配色铁律
1. 每页 ≤ 3 种颜色（不含黑白灰）
2. 低饱和度更专业稳重
3. 对比度 ≥ 4.5:1（WCAG 2.2 AA 标准）
4. 深灰 #333333 > 纯黑 #000000
5. 暖色→创意/营销，冷色→商务/技术

## 三、字体搭配

### 推荐配对

| 标题字体 | 正文字体 | 风格特征 |
|----------|----------|----------|
| Garamond | Calibri | 经典而现代 |
| Arial Bold | Arial | 企业标准，最安全 |
| Calibri Bold | Calibri | 清晰现代（默认首选） |
| Georgia | Calibri | 知性学术风 |
| Trebuchet MS | Calibri | 清爽商务 |

### 字号层级
- 标题：36-44pt Bold
- 章节标题：20-24pt Bold
- 正文：14-16pt
- 说明文字：10-12pt

### 字体铁律
- ✅ 最多2种字体（标题+正文）
- ✅ 单字族多字重（Regular/Medium/Bold）
- ❌ 不使用纯黑 #000000
- ❌ 不混搭风格迥异的字体

## 四、版式与布局

### 网格系统
- 推荐 **8pt 网格**：所有元素尺寸和间距为8的倍数
- 边距 ≥ 0.5 英寸
- 内容块间距 0.3-0.5 英寸
- 留白 ≥ 页面 50%

### 阅读路径
| 类型 | 适用场景 | 说明 |
|------|----------|------|
| Z型 | 信息型页面 | 左上→右上→左下→右下 |
| F型 | 文字型页面 | 上方横读→左侧纵读 |
| 居中对称 | 重要声明 | 焦点居中，四周留白 |

### 常见版式模板
1. **两栏式**：文字左+配图右（或反之）
2. **卡片式**：圆角矩形分组展示内容
3. **2×2/2×3网格**：多要点平铺
4. **全出血图片**：图片占满一侧+文字覆盖
5. **Bento Grid**：模块化不对称卡片排列（2026趋势）

## 五、数据可视化

### 图表选择
- **柱状图**：比较不同类别
- **折线图**：展示时间趋势
- **饼图**：展示比例（慎用，≤3个分段）
- **散点图**：展示相关性
- **单一大数字**：最强冲击力（如"42,000 活跃用户"）

### 图表美化
1. 去装饰化：去除网格线、阴影、3D效果
2. 颜色引导：重点数据高亮，其余柔和
3. 直接标注：数据标签放在图表上而不是图例
4. 每图一个核心洞察

## 六、叙事结构

### 标准 PPT 结构
```
封面 → 目录 → 正文 → 总结 → CTA
                    ↓
         三幕剧：设定(20%) → 冲突(60%) → 解决(20%)
```

### 叙事弧线
```
问题 → 解决方案 → 证明 → 诉求（CTA）
```

### 常见类型长度建议
| 类型 | 推荐页数 |
|------|----------|
| 投资者 Pitch | 10-14 页 |
| 销售演示 | 12-20 页 |
| 内部汇报 | 8-15 页 |
| 学术报告 | 15-25 页 |

## 七、2025-2026 设计趋势

1. **编辑风 Editorial**：大字体+高质量图片+精简文字
2. **Bento Grid**：模块化卡片布局
3. **AI 配图**：Copilot / gpt-image-2 生成定制配图
4. **3D 视觉**：3D图标和插图增加深度
5. **手机优先**：文字更大，布局更简
6. **数据故事化**：去装饰化图表
7. **极简为主，极繁为辅**
