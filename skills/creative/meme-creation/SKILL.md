# 🎨 表情包制作技能 (Meme Creation) v1.0

> **核心认知**：2025年，表情包制作已从「设计问题」变为「工具驱动+流程工程问题」。
> 不需要美术功底，只需要执行力。

## 一、核心流程

### 静态表情包制作
```
确定IP形象 → AI批量生图(网格流) → 高清放大+切片 → 批量抠图
→ 加文字 → 压缩 → 上架/分享
```

### 动态表情包制作（额外步骤）
```
静态图 → AI视频生成(2-3秒, 5fps) → 转GIF → 压缩<500KB → 上架
```

## 三、关键技巧

### 矩阵网格流（最推荐的批量生成方式）
1. 选择模型：Nano Banana Pro（画风稳定）
2. 5×5 网格图一次性生成 25 张
3. Prompt 中强制：
   - **绿幕背景**：`Solid Green Background #26ed16`（方便后续一键抠图）
   - **粗黑描边**：`Thick black outline`（更像贴纸、边缘干净）
4. 先放大（1024→2048px）再 5×5 切片，确保清晰度
5. 批量抠图去背景

### 动态表情包技巧
1. 时长控制在 **2-3秒**
2. 帧率 **5fps**（平衡体积和流畅度）
3. 分辨率 **240×240**（微信标准）
4. 文件大小 **<500KB**（微信限制）
5. 加 **2px 白色描边**（微信规范）

### ⚠️ 绿幕溢色陷阱
AI生成的绿色背景会产生**环境溢色**（绿色反射到人物边缘）。
**正确做法**：先用AI生成无背景图 → PS手动抠图 → **手动填充纯绿色** → 再丢给动态生成工具

## 四、Prompt 模板

### 网格流 Prompt 模板
```
根据 参考图片中展示的角色，创作一套色彩丰富、手绘风格的 LINE 式半身 Q 版表情包肖像，
并确保其头部配饰描绘准确。将这些肖像以 5x5 网格形式排列，
并配上常见的聊天短语或相关的幽默梗。文字请使用手写风格字体。
输出必须是原创的，请勿直接复制参考图片。最终图像应为 4K 分辨率。
Solid Green Background #26ed16. Thick black outline.
```

### 动态表情 Prompt 模板
```
生成【附件中的{角色}嘴角上扬，眼神狡黠，眨一下眼睛(Wink)。
动作幅度夸张，身体跟随笑声轻微晃动，画面生动有趣，
高品质渲染，纯色背景，循环动画，表情包素材。】
比例 1:1，时长: 2s，帧率：5fps 的 GIF
```

### 文案生成 Prompt
```
请帮我生成24个不同的有创意的表情包提示词，涵盖以下情绪/场景：
{你的情绪列表}
表情包里要包含对应的文字，文字风格{网感/可爱/沙雕等}
```

## 五、微信上架规范

### 素材清单
| 素材 | 尺寸 | 格式 | 大小限制 |
|------|------|------|---------|
| 表情主图 ×24 | 240×240 | PNG/GIF | <500KB |
| 缩略图 ×24 | 120×120 | PNG | <100KB |
| 封面图 | 240×240 | PNG | <500KB |
| 聊天图标 | 50×50 | PNG | <80KB |
| 详情横幅 | 750×400 | PNG/JPG | <500KB |

### 微信审核避坑
- ✅ 风格统一 > 画得好看（审核最怕风格不一致）
- ❌ 不要遮挡表情主体的脸部
- ❌ 边缘抠图要干净，不留白点/绿边
- ❌ 必须正方形（长方形会被自动裁剪）
- ❌ 描述不要带广告、外链、营销导向
- ✅ 宁愿裁小不要拉伸
- ⚠️ 涉及人物肖像需授权书

## 七、常见问题

### Q: 不会画画能做表情包吗？
A: 完全可以。2025年AI工具已将此门槛降到零。你只需要选择风格和写提示词。

### Q: 动态表情包为什么总超500KB？
A: 控制：时长≤2秒、帧率5fps、分辨率240p。用 docsmall 或 ezgif 压缩。

### Q: 如何让AI生成的角色保持一致？
A: 使用网格流一次性生成（同一Prompt+同一模型保证风格统一）。或者使用垫图（IP三视图）作为参考。

### Q: 上架后多久能通过审核？
A: 一般 1-3 天。风格统一、尺寸合规的前提下通过率很高。
