# UI 风格深度指南

来源：第 1 轮广度扫描 + 第 2 轮深度挖掘 综合整理

## 扁平化设计 (Flat Design)
- **根源**：瑞士风格(Swiss Style) + 包豪斯(Bauhaus) → 微软 Metro → iOS 7 全面引爆
- **核心特征**：无阴影/渐变/纹理、二维平面、鲜明色彩、清晰排版、极简
- **进化**：扁平化 1.0(纯粹扁平) → 扁平化 2.0(微阴影+渐变) → 融入 Material Design
- **优点**：加载快、响应式友好、开发成本低、跨设备一致
- **缺点**：按钮可见性不足、导航时间增加 22%（NN/g 研究）
- **最佳实践**：使用扁平 2.0 手法（微阴影区分层次），注意 CTA 按钮对比度

## Material Design 3
- **起源**：Google 2014 → Material You 2021 → M3 Expressive 2025
- **三大子系统**：Color / Typography / Shape
- **Material You 核心**：动态配色（从壁纸提取 → 65 种颜色）
- **M3 Expressive 核心**：情感设计模式、更大按钮、弹性动效、35 种形状
- **关键数据**（Google 46 项研究，18,000+ 参与者）：
  - 视觉定位速度提升 4 倍
  - 87% 的 18-24 岁用户偏好
  - 消除年龄差距
  - 品牌现代感 +34%
- **组件体系**：Action/Containment/Navigation/Selection/Text Input 五类
- **无障碍**：动态配色内置 WCAG 对比度要求

## 玻璃拟态 (Glassmorphism)
- **提出**：Michal Malewicz 2020，macOS Big Sur 推广
- **四大要素**：半透明 + 背景模糊 + 鲜艳背景 + 精细边框
- **2025 浏览器支持**：全球 ~96%
- **适用场景**：桌面系统、创意落地页、重点卡片、导航面板
- **不适用**：按钮/开关等交互元素、大面积全屏、移动端全屏
- **性能**：每页 ≤3 个元素，blur 8-16px

## 新拟物化 (Neumorphism)
- **提出**：Alexander Plyuto 2019
- **原理**：单一光源(左上→右下) + 内外阴影 + 同色系
- **争议**：低对比度导致严重可访问性问题
- **适用**：仅局部点缀（计算器、音乐播放器），与其他风格混用
- **不适用**：大面积使用、信息密集型应用、严肃场景

## 粗野主义 (Brutalism)
- **建筑起源**：20 世纪中期 → 数字复兴 2010s
- **核心**：结构外露、功能>形式、反精致
- **特征**：硬阴影、粗黑边框、高对比色、系统字体
- **适用**：创意机构、品牌差异化、信息密集型数据界面
- **代表**：Linear、Notion、Arc Browser（结构层面）
